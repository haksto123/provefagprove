# Prøvefagprøve - Håkon Stokkenes 
# Målet for oppgaven: 

Målet for oppgaven er å lage en quiz app der brukere kan svare på spørsmål med flere valgmuligheter, i tillegg skal brukeren se sin poengsum ved slutten. Spørsmål og metadata ivaretas på ein server med et dokumentert intergrasjonsslag mellom klient og server. Appen skal også ha funksjon for å starte quizen på nytt. Det skal tas hensyn til grunnleggende prinsipper for universell utforming, som lesbarhet, kontraster og enkel navigasjon. 

# Fremgangsmåte: 

Eg tenker å sjå gjennom krav, planlegge sikkerthet, og sette opp datamodell.  

link til datamodell  

(kan hende denne blir endret iløpet av utviklings prosessen) 

Lage tabeller, følge datamodell eg har laget. Fikse roller, triggers, og views for sikkerthet. 

Opprette ny app, fikse UI  

Lage stored procedures/views for å hente data ifra backend (SQL Server) inn til front end (vue) 

Fin pussing av design og funksjonalitet 

Legge til ekstra funksjonalitet viss eg har ekstra tid til overs
